# 📂 DropFilesKgpk — Личный файлообменник

**DropFilesKgpk (DFK)** — это полнофункциональный веб-сервис для загрузки, хранения и обмена файлами. Полностью бесплатный, с открытым исходным кодом и системой аккаунтов.

> 🌐 **Попробовать:** [dropfileskgpk.vercel.app](https://dropfileskgpk.vercel.app)

---

## ✨ Возможности

### � Аккаунты и безопасность
- Регистрация и вход через email/пароль
- Подтверждение email через 6-значный код
- **Двухфакторная аутентификация (2FA)** — дополнительный код при входе
- Пароли хранятся в зашифрованном виде (bcrypt)
- Восстановление пароля через email
- Квота хранилища на каждого пользователя

### 📂 Файлы
- **Загрузка любых файлов** — документы, архивы, изображения, видео, исполняемые файлы и т.д.
- Drag-and-drop перетаскивание
- **Загрузка целых папок** — одним кликом
- Переименование и удаление файлов
- Превью изображений (полный размер в модальном окне)
- Красивые иконки по типу файла (50+ расширений)
- Мгновенный поиск по имени файла

### � Гибкий доступ (как Google Drive)
| Тип доступа | Описание |
|---|---|
| 🔒 **Приватный** | Файл виден только владельцу |
| 🔗 **По ссылке** | Секретная ссылка с токеном — знает ссылку, может скачать |
| 🌐 **Публичный** | Любой может найти и скачать файл |

- Ссылки с настраиваемым сроком жизни
- Лимит скачиваний на ссылку
- Мгновенный отзыв ссылок
- Копирование ссылки одним кликом

### 📁 Папки
- Создание вложенных папок
- Навигация по структуре папок
- Удаление папок вместе с содержимым

### 👨‍💼 Админ-панель
- Список всех пользователей
- Изменение лимита хранилища для каждого пользователя
- Удаление пользователей
- Статистика файлов

### 🎨 Интерфейс
- Тёмная и светлая тема с плавным переключением
- Адаптивный дизайн (мобильные, планшеты, десктоп)
- Toast-уведомления о действиях
- Статистика на дашборде (файлы, папки, публичные, размер)
- Красивая пустая страница с подсказками

### 📄 Юридическая часть
- Страница «Политика конфиденциальности» по ФЗ-152
- Информация об операторе, целях, правах пользователей

| Компонент | Стек |
|---|---|

| Язык | TypeScript |

## 🚀 Быстрый старт (локально)

Для локальной разработки проект использует SQLite. Для продакшена — PostgreSQL через Neon.

```bash
# 1. Клонирование
git clone https://github.com/freska46/dropfileskgpk.git
cd dropfileskgpk

# 2. Установка зависимостей
npm install

# 3. Настройка окружения
# Скопируйте .env и заполните переменные
# Для локального тестирования достаточно оставить значения по умолчанию

# 4. Запуск
npm run dev
# → http://localhost:3000
```

## 🌐 Деплой

Проект развёрнут на **Vercel** с использованием:
- **Neon** — бесплатная PostgreSQL база данных (500 МБ)
- **Supabase Storage** — бесплатное хранилище файлов (1 ГБ)
- **Gmail SMTP** — отправка кодов подтверждения

Все переменные окружения настраиваются через панель Vercel.

## 📋 API Endpoints

### Аутентификация
- `POST /api/auth/register` — регистрация
- `POST /api/auth/signin` — вход
- `POST /api/auth/forgot-password` — запрос кода сброса пароля
- `POST /api/auth/reset-password` — сброс пароля

### Файлы
- `GET /api/files` — список файлов
- `POST /api/files/upload` — загрузка
- `GET /api/files/:id/download` — скачивание
- `PATCH /api/files/:id` — переименование / перемещение
- `DELETE /api/files/:id` — удаление
- `POST /api/files/:id/share` — управление доступом

### Шаринг
- `GET /api/share/:token` — информация по ссылке
- `GET /api/share/:token/download` — скачивание по ссылке

### Прочее
- `GET/POST /api/settings` — настройки аккаунта
- `GET/POST/PATCH/DELETE /api/admin` — админ-панель
- `POST /api/two-factor` — управление 2FA
